# What Launch 🍽️

> "오늘 점심 뭐먹지?" 고민을 가볍게 해소하는 뽑기 게임

## 📋 프로젝트 소개

What Launch는 매일 반복되는 점심 메뉴 선택의 고민을 해결하기 위한 간단한 웹 애플리케이션입니다. 
위치와 음식 장르를 선택하면 주변 식당 중 하나를 랜덤으로 추천해드립니다.

## ✨ 주요 기능

- 📍 **위치 기반 검색**: 주소 또는 장소명을 입력하여 주변 식당 검색
- 🍜 **음식 장르 선택**: 일식, 한식, 중식, 양식 등 원하는 음식 장르 선택
- 🎲 **랜덤 선택 옵션**: 음식 장르도 랜덤으로 선택할 수 있는 옵션 제공
- 🗺️ **지도 시각화**: 
  - 입력한 주소 위치를 지도에 포인트로 표시
  - 랜덤으로 추천된 식당 위치를 지도에 포인트로 표시
- 🔄 **다시 선택하기**: 마음에 들지 않으면 언제든지 다시 추천받기 가능

## 📖 사용 방법

1. 웹 애플리케이션에 접속
2. 현재 위치(주소 또는 장소명) 입력
3. 음식 장르 선택
   - 원하는 장르 선택 (일식, 한식, 중식, 양식 등)
   - 또는 "랜덤" 선택으로 장르까지 랜덤하게 결정
4. "추천받기" 버튼 클릭
5. 지도에서 입력한 주소와 추천된 식당 위치 확인
6. 마음에 들지 않으면 "다시 선택하기" 버튼으로 재추천 가능
